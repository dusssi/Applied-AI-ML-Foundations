# 🧩 Assignment 1 — K-Means Customer Segmentation

---

## 🎯 Objective

Segment customers based on Annual Income, Spending Score, and Age to identify distinct customer groups.

---

## 🛠️ Techniques Used

* K-Means Clustering
* Elbow Method (Optimal Cluster Selection)
* Silhouette Score (Cluster Quality Evaluation)
