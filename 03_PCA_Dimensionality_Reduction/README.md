# Assignment 3 - PCA (Dimensionality Reduction)

## Objective
Reduce high-dimensional image data using PCA.

## Techniques Used
- Principal Component Analysis (PCA)
- Explained Variance
- Dimensionality Reduction

## Output
- 2D visualization of digits
- Reduced dataset to 50 dimensions
- Optimal components for 95% variance

## Key Insight
PCA reduces dimensionality while preserving maximum information, improving efficiency.
