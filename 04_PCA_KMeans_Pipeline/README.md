# Assignment 4 - PCA + KMeans Pipeline

## Objective
Evaluate clustering performance before and after applying PCA.

## Techniques Used
- PCA (Dimensionality Reduction)
- KMeans Clustering
- Pipeline (sklearn)
- Adjusted Rand Index (ARI)

## Output
- Comparison of clustering performance
- Visualization of clusters

## Key Insight
PCA can improve clustering by removing noise, but may also reduce accuracy if too much information is lost.
