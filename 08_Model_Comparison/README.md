# Assignment 8 - Model Comparison

## Objective
Compare multiple machine learning models on classification task.

## Models Used
- Logistic Regression
- SVM (RBF Kernel)
- Random Forest

## Evaluation Metrics
- Accuracy
- Precision
- Recall
- F1 Score
- AUC-ROC

## Techniques Used
- Stratified Cross Validation
- ROC Curve Comparison

## Output
- Performance comparison table
- ROC curves for all models

## Key Insight
Model performance varies depending on data; no single model is universally best.
